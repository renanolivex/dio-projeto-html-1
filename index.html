O que são Tags HTML?

Tags HTML são basicamente instruções passadas para o HTML que funcionam para a formatação de textos, criação de links e outros. Como é dito pela Host Gator:

"É uma linguagem de marcação que define a estrutura e o conteúdo das páginas da web. As tags HTML são os blocos de construção fundamentais do HTML, usadas para estruturar o conteúdo de uma página."

A Tag costuma ser iniciada por "<", seguida pela tag escolhida e fechada por ">" e finalizada por "<" "/" tag escolhida e novamente fechada por ">"

Exemplos de Tags

1. Tags "h1 até "h6"

São tags utilizadas em títulos iniciadas por "h" e um número de 1 a 6 sendo 1 o título com o maior tamanho de 6 com o de menor tamanho.

Segue abaixo exemplos:

Tag < h1 > Exemplo escrito: < h1 > Título 1 < /h1 > Resultado:

Título 1

Tag < h2 > Exemplo escrito: < h2 > Título 2 < /h2 > Resultado:

Título 2

Tag < h3 > Exemplo escrito: < h3 > Título 3 < /h3 > Resultado:

Título 3

Tag < h4 > Exemplo escrito: < h4 > Título 4 < /h4 > Resultado:

Título 4

Tag < h5 > Exemplo escrito: < h5 > Título 5 < /h5 > Resultado:

Título 5

Tag < h6 > Exemplo escrito: < h6 > Título 6 < /h6 > Resultado:

Título 6

2. Tag "p"

Em seguida temos outra tag muito utilizada para separar parágrafos o < p >.

Segue exemplo escrito: < p > Esse é um parágrafo feito por uma tag "p". < /p >

Resultado: Esse é um parágrafo feito por uma tag "p".

3. Tag "mark"

A tag mark é usada para marcar alguma palavra ou frase, como se fosse uma caneta marca-texto (lumicolor).

Exemplo escrito: Essa < mark > palavra < /mark > está marcada.

Resultado: Essa palavra está marcada.

4. Tag "small"

A tag small serve para diminuir o tamanho das letras que envolvem a tag

Segue exemplo escrito: Essa < small > palavra < /small > foi minimizada pela tag small.

Resultado: Essa palavra foi minimizada pela tag small.

5. Tag "i"

A tag i é usada para transformar um texto em italico.

Segue exemplo escrito: Essa < i > palavra < /i > está escrita em italico.

Resultado: Essa palavra está escrita em italico.

6. Tag "u"

A tag i é usada para grifar um texto.

Segue exemplo escrito: Essa < u > palavra < /u > está grifada.

Resultado: Essa palavra está grifada.

7. Tag "strong"

A tag strong é usada para mudar um texto para negrito.

Segue exemplo escrito: Essa < strong > palavra < /strong > está em negrito.

Resultado: Essa palavra está em negrito.

8. Tags "ol", "ul" e "li"

As tags ol, ul e li são usadas para fazer "listas". A tag li geralmente é usada dentro das tags ul e ol sendo itens dentro das listas. O que difere as tags ul e ol é que a tag ul faz uma lista não ordenada enquanto a tag ol cria uma lista ordenada

Segue exemplo escrito da combinação de tags ul e li: Essa é uma lista de frutas não ordenada: < ul >< li >Banana< /li >< li >Maçã< /li >< li >Pera< /li >< /ul >

Resultado: Essa é uma lista de frutas não ordenada:

  • Banana
  • Maçã
  • Pera

Segue exemplo escrito da combinação de tags ol e li: Essa é uma lista de frutas ordenada: < ol >< li >Banana< /li >< li >Maçã< /li >< li >Pera< /li >< /ol >

Resultado: Essa é uma lista de frutas ordenada:

  1. Banana
  2. Maçã
  3. Pera

9. Tag "a"

A tag a é usada para criar links (palavras ou textos clicáveis) que costumam enviar o usuário a outra página, local do site ou até mesmo uma aba no navegador. Para adicionar o local que o link enviará o usuário é necessario utilizar também "href="local"" após o "a".

Segue um exemplo escrito: Essa < a href="https://www.dio.me/" > palavra < /a > levará para o site da DIO.

Resultado: Essa palavra lavará para o site da DIO.

11. Tag "sub"

A tag sub é usada para escrever um texto com letras menores e com alinhamento abaixo dos textos normais.

Segue um exemplo escrito: Essa < sub > palavra < /sub > tem um tamanho menor que as outras e alinhamento para baixo.

Resultado: Essa palavra tem um tamanho menor que as outras e alinhamento para baixo.

12. Tag "sup"

A tag sup, como a tag sub, também diminui o tamanho do texto escrito, a diferença é que o alinhamento do texto faz com que fique para cima dos textos normais.

Segue um exemplo escrito: Essa < sup > palavra < /sup > tem um tamanho menor que as outras e alinhamento para baixo.

Resultado: Essa palavra tem um tamanho menor que as outras e alinhamento para cima.

13. Tag "blockquote"

A tag blackquote é usada quando se está citando algum texto. O texto fica com espaçamento no inicio.

Segue um exemplo escrito: O texto a seguir foi retirado do site de exemplos de blockquote: < blockquote > "Esse texto é um exemplo de Blockquote. < /blockquote >

Resultado: O texto a seguir foi retirado do site de exemplos de blockquote:

"Esse texto é um exemplo de Blockquote."

14. Tag "font"

A tag font é usada para colorir textos de acordo com a cor escolhida.

Segue um exemplo escrito: Essa < font color="blue" > palavra < /font > está colorida por causa da tag font.

Resultado: Essa palavra está colorida por causa da tag font.

15. Tag "del"

A tag del serve para indicar algo que foi "excluido", o texto fica com um risco no meio.

Segue um exemplo escrito: Essa < del > palavra < /del > foi "deletada".

Resultado: Essa palavra foi "deletada"

16. Tag "abbr"

A tag abbr funciona como um tipo de explicação para algo. Ao colocar o mouse em cima do item que se encontra o abbr, será mostrada oque foi escrito no abbr

Segue um exemplo escrito: A sigla escolhida como exemplo é a < abbr title="HyperText Markup Language" >HTML.< /abbr >.

Resultado: A sigla escolhida como exemplo é a HTML. (necessário colocar o mouse em cima para ver o exemplo)
